# 60. 개인정보보호·컴플라이언스 스펙

> 대한민국 **개인정보 보호법(PIPA)** 및 관련 고시 기반. 본 제품은 생년월일·출생시·출생지(대략) 등 **개인식별 가능 정보**와 **민감정보에 해당할 수 있는 성향 응답**을 수집하므로, 수집·보관·파기 전 과정을 기록·검증할 수 있어야 한다.

---

## 0. 전제와 리스크

- 생년월일·출생시는 **개인정보 보호법상 개인정보**에 해당.
- 성격 검사 응답(MBTI/Enneagram) 자체는 민감정보 분류에 직접 해당하지 않으나, 조합에 따라 **프로파일링** 성격이 강하므로 동등 수준으로 보호.
- 종교·사상·건강 등 **민감정보**(PIPA §23)는 수집하지 않는다. 사주·자미두수 해석 결과는 **추론 데이터**로 분류하되, 본인 외 제3자 제공 금지 원칙 준수.
- **14세 미만 회원 가입 불허** (법정대리인 동의 확보 복잡성 회피). 가입 시점 생년 기반 검증.

## 1. 수집 항목·목적·보유기간

| 항목 | 수집 방법 | 목적 | 보유 기간 |
|---|---|---|---|
| 이메일 | 회원 가입 | 인증·로그인·공지 | 회원 탈퇴 시까지 + 30일 버퍼 |
| 닉네임 | 회원 가입 | 리포트 표시 | 동상 |
| 생년월일·생시·출생지 | 프로필 생성 | 사주·자미두수 엔진 계산 | 프로필 삭제 시까지 |
| 성향 설문 응답 (MBTI·Enneagram) | 설문 | 엔진 계산 | 프로필 삭제 시까지 |
| 결제 정보(빌링키) | 결제 | 정기 청구 | 구독 만료 후 90일 |
| 접속 로그(IP 해시) | 자동 | 보안·부정행위 탐지 | 3개월 |
| 쿠키·디바이스 식별자 | 자동 | 세션·분석 | 세션/1년 |

**별도 동의 대상:**
- (필수) 서비스 이용약관
- (필수) 개인정보 수집·이용 동의
- (필수) 민감 추론 결과 생성 동의 (사주·자미두수 기반 성향 추론)
- (선택) 마케팅 정보 수신 동의

## 2. 동의 관리

`app.consents` 테이블([01-data-model §5.8](./01-data-model-v0.1.md))에 append-only 기록.

### 2.1 기록 시점
- 회원 가입
- 약관 개정 시 재동의
- 마케팅 동의 on/off
- 민감 추론 결과 생성 토글

### 2.2 보관 정보
- 동의 시각, 약관 버전, IP 해시, UA 해시 (IP·UA 원문 저장 ❌)
- 철회 이력

### 2.3 사용자 열람
- `/settings/privacy` 페이지에서 자신의 동의 이력 열람 + CSV 다운로드
- 철회·재동의 원클릭

## 3. 사용자 권리(§35~§37)

### 3.1 열람권 (PIPA §35)
- `/settings/privacy/my-data` → 본인 데이터 JSON 내보내기
- 포함: profile(들), engine_results, survey answers, consents, purchases
- 제외: 타인의 데이터, 내부 로그

### 3.2 정정·삭제 (§36)
- 프로필 수정: 즉시 반영
- 계정 삭제: 요청 즉시 soft delete → **30일 유예** → 크론으로 hard delete
- 일부 항목(결제 내역, 감사 로그)은 법정 보존 기간 동안 격리 저장 후 파기

### 3.3 처리정지 (§37)
- "마케팅 수신 거부": 즉시
- "분석·추론 정지": 구독 상태 유지하되 해석 생성 중단 (선택지 명시)

### 3.4 응답 기한
- 법정 10일 이내. 내부 SLA: **5영업일**.
- 요청은 이메일(privacy@arche.app) + 웹 폼 모두 수용.

## 4. 처리 흐름도

```
사용자 입력
  │
  ▼
클라이언트 (TLS 1.3)
  │
  ▼
Vercel Edge (CDN) — 인증 쿠키 검증
  │
  ▼
Node Function (tRPC)
  │
  ├─► Postgres (Supabase) — 필드 암호화(빌링키 등)
  ├─► KV (Upstash Redis) — 캐시/레이트리밋 (PII 저장 ❌)
  ├─► Sentry — 에러 로그 (PII 스크러빙)
  ├─► PostHog — 행동 이벤트 (이메일/이름 전송 ❌)
  └─► Anthropic LLM — 해석 생성 (PII 미전송, 추상 결과만)
```

모든 인프라는 **한국 리전 우선** (Supabase Seoul, Vercel Seoul). PostHog·Sentry는 EU 리전. LLM은 미국 리전.

## 5. 국외 이전

일부 서비스는 국외 서버를 이용하므로 **국외 이전 고지** 필수. 약관에 다음 표 기재:

| 수탁자 | 국가 | 항목 | 목적 | 보유기간 |
|---|---|---|---|---|
| Vercel Inc. | 미국·한국 | 접속 로그·쿠키 | 호스팅·CDN | 90일 |
| Anthropic PBC | 미국 | 엔진 출력(추상), 프롬프트 | 해석 문장 생성 | 30일(Anthropic 정책) |
| Stripe Inc. | 미국·국제 | 결제 식별자 | 국제 결제 | 각 약관 |
| Sentry / PostHog | 미국·EU | 에러·이벤트 (익명화) | 모니터링 | 90일 |
| Resend | 미국 | 이메일 주소 | 트랜잭션 메일 | 30일 |

**이메일·이름·생년월일 원본이 국외 수탁자에 전달되는 경우는 없다** — LLM은 추상화된 엔진 출력만 받는다.

## 8. 이용약관

`/legal/terms`에 게시. 포함 항목:
- 서비스 정의·제공 내용
- 이용계약 성립
- 유료 서비스·결제·환불
- 이용 제한·해지
- 책임 제한(해석 결과의 오락·자기 탐구 목적 명시)
- 분쟁 해결(대한민국 법·관할 법원)

**중요 면책 문구:**
> 본 서비스에서 제공하는 사주·자미두수·성향 분석 결과는 자기 이해와 탐구를 돕기 위한 참고 자료입니다. 본 결과는 의료·법률·투자·진로 결정의 근거가 아니며, 당사는 결과의 활용에 따른 어떠한 법적 책임도 지지 않습니다.

## 9. 아동 보호

- 가입 시 생년 입력 → 현재 한국 나이 14세 미만 차단
- 생년월일이 14~18세인 경우: 가입은 허용하되 **결제 불가**, 부모 동의 안내(향후)
- 프로필 피검사자가 아동일 수 있음(부모가 아이 사주를 등록) → 수집·처리 목적·보관 기간 별도 고지

## 11. 쿠키 정책

| 쿠키 | 용도 | 수명 | 제3자 |
|---|---|---|---|
| `__Secure-auth.session-token` | 세션 | 세션 | 자사 |
| `__Host-auth.csrf-token` | CSRF | 세션 | 자사 |
