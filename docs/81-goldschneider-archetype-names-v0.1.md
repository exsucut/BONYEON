# 81. Goldschneider 48주 원형 네이밍 워크시트

> Goldschneider 원서의 네이밍·서술문은 **차용하지 않는다**([12-engine-goldschneider §0](./12-engine-goldschneider-spec-v0.1.md)). 본 워크시트는 전통 점성학의 별자리·데케이트 특성(공유 지식)에 기반해 **자체 저술한** 한글·영문 네이밍이다. 최종 확정 전 변리사·법무 리뷰 1회 필요.

---

## 규칙

1. 각 주 1개 ID(1~48), `weekKey`, 날짜 범위, 별자리·데케이트
2. 한글명: **"~자/~가/~인"** 명사형 자제, 추상 명사로 ("돌파자" ✗ / "돌파의 축" ✓)
3. 영문명: `The [Noun]`. 원서 표현(The Dreamer, The Pioneer 등) **금지**
4. 핵심 키워드: 3~5개. 강점·긴장 균형
5. 성장축: 1문장, "~에서 ~로" 형태
6. 본 워크시트 내 중복어휘 피하기

## 검수 체크리스트

- [ ] 모든 한글·영문명이 Goldschneider 원서·한국어 번역본의 명칭과 **직접 유사도 없음** 확인
- [ ] 각 주의 **3~5 키워드**가 전통 점성학 공유 지식 범위 (원서 문장을 번역하지 않음)
- [ ] 변리사 1차 유사도 체크 (저작물 vs 저작물)
- [ ] 톤 일관성: 한글은 명사구 중심, 영문은 The + Noun
- [ ] 스크린 리더 발음 확인 (한자 없음)
