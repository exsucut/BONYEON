## 1. 브랜드 포지션

### 1.1 포지셔닝 스테이트먼트

> **20~40대의 지적 호기심 있는 사용자**에게,
> **운명은** 사주·자미두수·별자리·MBTI·애니어그램 **5개 체계가 한 사람을 어떻게 겹쳐 설명하는지**를 **모던하고 미니멀한 인터페이스로** 보여주는 **분석 서비스**다.
>
> 경쟁 서비스(점신·포스텔러·헬로우봇)와 달리, 우리는 **단일 체계의 예언자가 아니라 여러 체계를 중립적으로 병치하는 에디터**다. 운세가 아니라 **자기 이해**.

### 1.2 한 문장 테스트

- "이것이 우리의 브랜드이다"를 설명할 때 **한 문장으로 가능한가?**
  - ✅ "여러 체계로 나를 겹쳐 보는 서비스."
  - ❌ "전통 명리와 현대 심리학을 융합하여 사용자에게 맞춤형 인사이트를 제공하는 프리미엄 라이프스타일 플랫폼."

### 1.3 브랜드가 아닌 것

| 우리는 아니다 | 이유 |
|---|---|
| 점집 | 예언을 팔지 않는다. 사용자가 스스로 해석할 재료를 준다. |
| 힐링 앱 | 감정 위로가 목적이 아니다. 인식·이해가 목적. |
| SNS | 공유는 있으나 부차적. 프로필 공개·팔로우 없음. |
| 연애 매칭 | 궁합 분석은 있지만 매칭·채팅 없음. |
| 종교·영성 커뮤니티 | 신비주의 톤 배제. 세속적 지적 톤. |

## 2. 브랜드 키워드

### 2.1 5개 코어 속성

| 속성 | 정의 | 시각적 번역 | 언어적 번역 |
|---|---|---|---|
| **Lucid** (명료한) | 용어를 피하지 않고 정확히 쓰되 설명한다 | 충분한 여백, 선명한 계층 | 한자 사용 OK, 괄호로 풀이 |
| **Editorial** (에디토리얼) | 잡지처럼 읽힌다. 카피 하나하나가 쓴 것 | 세리프 헤드라인, 좌측 정렬, 큰 타이포 | 짧은 단문, 한 문단 1아이디어 |
| **Composed** (절제된) | 과하지 않다. 아이콘·이모지·그라데이션 남발 안 함 | 흑백+포인트 1색, 플랫 아이콘 | 감탄사·느낌표 최소화 |
| **Credible** (신뢰성) | 근거를 보여준다 | 계산 trace 열기 버튼, 출처 각주 | "~라고 합니다" 대신 "~입니다" |
| **Curious** (호기심) | 확정하지 않고 제안한다 | 대비보다 병치, 불릿 대신 문장 | "당신은 X" 대신 "X 측면이 있습니다" |

### 2.2 비(非)-키워드

의도적으로 버리는 인접 속성:
- **Warm / Cozy** — 힐링 톤은 포스텔러·헬로우봇의 몫.
- **Playful / Cute** — 캐릭터·이모지 중심은 안 한다.
- **Mystical / Esoteric** — 별자리 앱의 보라-골드 그라데이션 금지.
- **Aggressive / Bold** — 핫토시·광고톤 아님.
- **Luxurious / Elite** — 골드·대리석·세리프 과장 금지.

### 2.3 톤 스펙트럼 위에서의 위치

```
        장난스러움                             엄숙함
        │                                         │
        │                       [우리]            │
        ├──────────────────────────●──────────────┤
        │                                         │
        [헬로우봇]                                [점신·원광만세력]
        [포스텔러]
```

```
        감정적                                 분석적
        │                                         │
        │                              [우리]     │
```

## 3. 브랜드명 후보

### 3.1 네이밍 요건

- **발음 3음절 이하** (한글/영문 공히 짧게)
- **한국어 비원어민도 발음 가능** (i18n 대비)
- **도메인 `.com` 또는 `.io` 획득 가능**
- **KIPO 상표 검색(45류 중 9/41/42류) 선점 없음**
- **부정적 2차 의미 없음** (특히 한·영·일·중)
- **검색 고유성** — 구글·네이버 1페이지에서 우리가 상단 노출 가능해야

### 3.2 네이밍 축(Axis)

사용자의 "모던/미니멀/지적" 톤을 네이밍 축으로 풀면:

- **축 A — 기하·광학 은유**: 5개 체계가 한 사람에게서 5개 굴절로 보이는 메타포. Prism·Facet·Lens·Mira 계열.
- **축 B — 좌표·지도 은유**: 자신을 좌표 위에 놓는 행위. Atlas·Meridian·Axis·Orbit 계열.
- **축 C — 한국어 단음절 추상**: 결·겹·터·온·틈 계열. 에디토리얼한 한글 브랜드.
- **축 D — 라틴·고어**: Sigma·Nuum·Aera·Alta 계열. 중립적 국제명.

### 3.6 최종 선정: **본연 (BONYEON)**

**의미.** 한자 **本然** — "본디 그러한, 타고난 대로의". 사람의 **근본 성향·진면목**.

- 사주의 **원국(原局)**·자미두수의 **명궁**·MBTI/Enneagram의 **유형**은 모두 "타고난 것"을 읽는 체계. "본연"은 이 다섯 체계가 가리키는 **공통 대상**이다.
- 한국어 2음절. 한국 타깃에 직관적 울림.
- 지적 톤: 한자어 기반, 경박하지 않고 단정.

**강점.**
- WebSearch 결과 SW·앱 영역에서 브랜드 크라우딩 **거의 없음**.
- `bonyeon.com`·`bonyeon.app`·`bonyeon.kr` 사전 미확인이지만 Latin 후보군보다 확보 가능성 높음.
- 한글 퍼스트 전략으로 Latin 포화 공간과 차별화.
- Mirror(반영)·The Pattern(패턴) 등 영문 경쟁자 대비 **문화적 정체성** 선명.

**약점·위험.**
- 사전 등재 일반어 → 단독 문자상표보다 **로고+지정상품 결합상표** 권장.
- 영문 로마자 `BONYEON` 7자, 해외 확장 시 가독 난도 중간.
- "본연식품" 등 식품 업계 일반어로 사용 사례 존재 — 9·42류는 깨끗할 가능성, KIPRIS 정식 검색 필수.

## 5. 워드마크·로고 방향

### 5.1 세 가지 길

네이밍 확정 후 로고는 아래 셋 중 하나로. **1-2순위를 가정하고 설계.**

**길 1. 커스텀 레터폼 워드마크** (권장)
- 기성 폰트에 약간의 튜닝을 더한 워드마크. Mailchimp·Medium·Headspace 방식.
- 외주 견적: 국내 100~300만원, 해외(Dribbble) $500~2000.
- 1인 자체 제작 가능성: Figma + Glyphs Mini. 약 40~80시간.

**길 2. 심볼 + 워드마크 조합**
- 심볼: 5엔진을 추상화한 마크(5점 도트, 5가지 굴절 선, 5겹 동심원 등).
- 공간 확장성 좋음. 파비콘·앱 아이콘 유리.
- 위험: 과욕하면 "영성 로고"로 빠짐. 절제 필요.

**길 3. 타이포그래피 워드마크 only** (가장 미니멀)
- 고품격 세리프 또는 지오메트릭 산세리프를 그대로 사용, 자간·크기 조정만.
- Arc·Vercel·Linear 방식. 제작 비용 최소.
- 권장 후보 폰트: **GT Sectra**, **Söhne**, **Editorial New**, **Neue Haas Grotesk**, (한글) **Pretendard**, **Sandoll Rix 고딕**, **산유화**.

### 5.2 로고 체크리스트 (결과물이 통과해야 할 테스트)

- [ ] 16×16px 파비콘에서 식별 가능
- [ ] 흑백 단색으로 인쇄 가능
- [ ] 3m 거리 모바일 화면에서 판독 가능
- [ ] 다크모드에서도 작동 (반전 시 시각 무너지지 않음)
- [ ] 애니메이션 친화 (최소 1가지 전환 모션 가능)
- [ ] 한국어/영어/SNS 아이콘 3버전 구비

### 5.3 심볼 컨셉 브레인스톰 (네이밍 확정 후 선택)

1. **다섯 줄의 굴절**: 프리즘을 통과한 빛이 5가닥으로 갈라지는 형상 (Facet/Mira에 적합).
2. **열 두 방위 원(圓)**: 12지 또는 12궁의 추상화, 그 위에 한 점 (Meridian/Atlas에 적합).
3. **겹친 원 5개**: 5엔진 = 5원의 벤다이어그램 (Facet/겹에 적합).
4. **한 획(一劃)**: 붓 한 획으로 그은 결. 한국미 계열. (결에 적합).
5. **별의 좌표 크로스헤어**: +자 십자선 + 작은 원 (Mira의 변광성 메타포).

### 5.4 로고 금지 목록

- ❌ 삼각형·피라미드·눈동자 — 오컬트·일루미나티 오해
- ❌ 별자리 선(constellation) 일러스트 — 포스텔러와 과도 유사
- ❌ 파스텔 그라데이션 — 톤 금지
- ❌ 한자 단일 (命·運·星 등) — 전통 점집 이미지

## 6. 보이스 앤 톤

### 6.1 6가지 글쓰기 규칙

**R1. 평서형 종결.** "~해요/~에요" 최소. "~입니다/~합니다" 기본.
- ❌ "당신의 사주는 신기하죠?"
- ✅ "당신의 일간은 금(金)입니다. 단단하고 의리를 중시하는 기질입니다."

**R2. 단정하지 말고 관찰하라.**
- ❌ "당신은 내향적인 사람입니다."
- ✅ "16유형 측정에서는 내향 축이 61%로 나왔습니다. 외부 자극보다 내적 정리가 에너지원입니다."

**R3. 한자어는 쓰되 괄호로 풀어라.**
- ✅ "일간(日干)이 경금(庚金)입니다."
- 독자를 똑똑하다고 가정하되, 처음 등장 용어는 한 번만 풀이.

**R4. 수량을 피하지 말라.**
- ❌ "에너지가 강하다"
- ✅ "오행 중 수(水) 비율이 40%로 가장 많습니다"

**R5. 대비보다 병치.** "A vs B"가 아니라 "A와 B".
- 단정을 피해 다원적 해석의 여지를 남긴다.

**R6. 1문단 1아이디어.** 긴 문단 금지. 스캔 가능한 리듬.

### 6.2 톤 매트릭스

| 컨텍스트 | 톤 | 예시 |
|---|---|---|
| 리포트 본문 | 에디토리얼, 평서 | "일간은 그 사람의 뼈대입니다. 당신의 뼈대는..." |
| 버튼·CTA | 간결한 명령형 | "프로필 만들기", "리포트 열기" |
| 오류 메시지 | 담담한 사과 | "생년월일을 다시 확인해 주세요." |
| 결제 화면 | 투명한 설명 | "월 9,900원, 언제든 해지할 수 있습니다." |
| 온보딩 | 짧은 질문 | "생년월일시를 아시나요?" |
| 이메일 | 편지처럼 | "안녕하세요. 이번 달 리포트가 준비되었습니다." |
| 마케팅 카피 | 담백한 선언 | "다섯 관점으로 당신을 겹쳐 봅니다." |

### 6.3 쓰지 말아야 할 어휘

| 금지어 | 이유 | 대체 |
|---|---|---|
| "당신은 ~한 사람" | 단정적 | "~한 측면이 관찰됩니다" |
| "운명", "팔자" | 결정론적 함의. 제품명을 본문에 쓰지 않는 편이 나음. | "사주 원국이 보여주는 것" |
| "신기하게도", "놀랍게도" | 엔터테인먼트톤 | 삭제 |
| "도사", "역술가" | 브랜드 톤 밖 | "명리 전통", "전통 해석" |
| 이모지 💫✨🔮 | 비주얼 랭귀지 영역 — 본문 금지 | 없음 |
| 감탄사 (와, 헉) | 톤 밖 | 없음 |

### 6.4 예시 비교 (점신 스타일 vs 우리)

**주제: 일간이 경금(庚金)인 사용자에게.**

_점신풍_: "당신의 일간은 경금! 강한 의지와 카리스마를 지닌 금속의 기운이 흐르고 있어요. 단단한 검처럼 결단력 있게 나아가는 스타일이죠. 다만 너무 강하면 부러질 수 있으니 유연함도 필요해요."

_우리풍_:

> **경금(庚金) — 단단함이 먼저 오는 사람.**
>
> 열 개의 천간 중 경(庚)은 양(陽)의 금(金)에 속합니다. 주조되지 않은 광물, 손질되기 전의 날(刃)입니다.
>
> 경금 일간의 사람은 첫 반응이 **판단**입니다. 상황을 만나면 좋다·나쁘다를 빠르게 가리고, 가린 뒤엔 흔들리지 않습니다. 이 기질은 결정권이 필요한 자리에서 진가를 발휘합니다.
>
> 동시에, 판단이 너무 먼저 오면 **공감**과 **대기**가 서투를 수 있습니다. 이 점은 MBTI 측정에서 T 축이 같이 강하게 나올 때 더 뚜렷합니다. 교차 해석 섹션에서 이어집니다.

## 9. 타이포그래피 개요

### 9.1 폰트 스택 1차 후보

### 9.2 폰트 선정 제약

- **라이선스 반드시 상용 허용** (Fraunces·Inter·Pretendard 모두 OFL, 상용 가능)
- 웹폰트 최대 2패밀리까지 (성능 예산)

### 9.3 타이포 원칙

- 세리프는 **디스플레이(헤드라인) 전용**. 긴 본문 금지.

## 11. 법적·상표 체크리스트

### 11.1 네이밍 확정 전 필수 체크 (각 후보별)

- [ ] KIPRIS 상표 검색 (키프리스 www.kipris.or.kr) — 9류·41류·42류
- [ ] USPTO TESS 검색 (해외 확장 대비)
- [ ] 도메인: `.com`, `.io`, `.app`, `.kr`, `.co.kr` 동시 확인
- [ ] SNS: Instagram, X, Threads, YouTube, TikTok 핸들
- [ ] Google 검색 1페이지 결과 점검
- [ ] 한·중·일·영 발음 시 부정어 충돌 없는지

### 11.2 콘텐츠 저작권

- **MBTI·Enneagram 공식 문항**은 권리가 있으므로 사용 불가. **자체 문항을 신규 작성**한다. 심리학 전공자 외주로 신뢰성 확보.
- **The Secret Language of Birthdays(Goldschneider)**: 48주 분류 체계는 아이디어로 보호 대상 아님. 다만 원서의 **각 일자별 해설 텍스트는 저작권**. 우리는 **자체 해설 신규 작성**. 원서 인용은 짧은 fair use 범위 내.
- **이미지·폰트**: 상용 라이선스 확인 후 사용. 무료 유출본 금지.

### 11.3 소비자 보호·표시 광고

- "100% 정확", "완벽한 예언" 등의 단정 금지 — 과장 광고 소지.
- 운세 결과로 인한 "결정"을 유도하는 카피 금지(예: "이 사람과 결혼하세요") — 의료·금융·법률 자문 아닌 엔터테인먼트임을 FAQ/약관에 명시.
