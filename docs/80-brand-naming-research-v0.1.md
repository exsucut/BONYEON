## 3. 한글 코인워드 후보 (신규 제안)

한국 타깃·Latin 공간 포화 상황을 고려한 **한국어 기반 브랜드**. 상표 크라우딩이 적고 일관된 문화적 정체성을 제공.

| 후보 | 의미 | 발음 난이도 | 해외 확장성 | 비고 |
|---|---|---|---|---|
| **오리 (五理)** | 다섯 이치 | 쉬움 | 낮음(Duck!) | ⚠️ "오리" 동음이의어(鴨) 치명적 |
| **오원 (五元)** | 다섯 근원 | 쉬움 | 중간 | "오원" 검색 결과 경쟁 존재 |
| **결** | 무늬·결 | 쉬움 | 낮음 | 기 조사에서 일부 검토. 한자 없음 |
| **격 (格)** | 틀·원형 | 쉬움 | 낮음 | 사주 격국과 직결 |
| **본연 (本然)** | 타고난 본성 | 쉬움 | 중간 | 의미 강함, 상업 상표 검색 필요 |
| **다섯결** | "다섯 결" | 쉬움 | 낮음 | 한국어 코인워드. 고유성 상 |
| **무지개** | 일곱 색 비유 | 쉬움 | 낮음 | 의미 확장 어려움(7색이지 5가 아님) |

### 3.1 한영 하이브리드 코인워드
문화적 뿌리를 한국어에서 두되, 글로벌 노출 가능한 형태.

| 후보 | 구성 | 음상 | 상표 리스크 |
|---|---|---|---|
| **PENTARA** | Penta + ARA(순우리말 "너울") | 모던 | 낮음 — 미검색 |
| **FIVRA** | Five + RA | 간결 | 낮음 |
| **AXIOME** | Axis + -ome (접미) | 과학적 | 중간 — 프랑스어 axiome(공리) |
| **ORIROOT** | 오리진 + Root | 직관 | 낮음 |
| **WONGUK** | 원국(原局) 로마자 | 도메인 특화 | 낮음(사주 용어) |

## 4. 시나리오 비교

### 시나리오 A — ARCHE 강행
- 도메인: `arche.kr`만 확보 or `getarche.com`
- 상표 출원: 9/41/42류 KIPRIS 즉시 출원 시도

### 시나리오 B — **한글 코인워드 (권장)**
- 후보: **본연 / 다섯결 / PENTARA** 중 상표 사전조사 후 선택
- 도메인: `.com`·`.kr` 동시 확보 가능성 높음
- 리스크: 최저
- 장점: 한국 타깃에 문화적 깊이, 디자인 정체성과 일치(차분·지적)
- 단점: 기 작성된 [70-brand-identity](./70-brand-identity-v0.1.md) §3~§4 개정 필요 (반나절 작업)

### 시나리오 C — 보류 + 신규 리서치
- ARCHE·백업 4종 모두 리스크 → 처음부터 네이밍 워크샵 재개
- 소요: 1주일 + 외부 네이밍 리뷰 의뢰(~₩500K)
- 리스크: P0 일정 지연(05-03 → 05-10)
- 장점: 최적해 탐색
